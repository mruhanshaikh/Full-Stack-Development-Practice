# Cart App (useReducer Demo)
# Live -: [shopingkartt.netlify.app](https://shopingkartt.netlify.app/)
A simple shopping cart project built with React to understand the useReducer hook.  
It demonstrates how state is managed using actions and a reducer instead of multiple useState functions.  
Users can add, remove, and clear items from the cart with a limit of 9 items.  
Focus of this project is learning centralized state management in React.
