React Form Cards Manager 
Live:https://dynamicprofilebuilder.netlify.app/

A  React project demonstrating two-way binding with multiple form fields, single change handler, input validation, and dynamic card rendering with delete functionality.

🚀 Features

Two-way binding for multiple inputs using a single state object

Single onChange handler to manage all form inputs

Validation using Object.entries()

Trimming whitespace to prevent empty or space-only values

Dynamic card rendering from submitted form data

Delete cards using Array.filter()

LocalStorage persistence to keep cards after page refresh

🧠 Concepts Practiced

Controlled Components in React

Two-way data binding

Handling multiple inputs with one state

Dynamic state updates using computed properties

Form validation

Immutable state updates

Array methods (map, filter)

LocalStorage integration
