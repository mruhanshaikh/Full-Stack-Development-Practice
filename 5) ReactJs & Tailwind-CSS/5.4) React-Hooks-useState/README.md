# React Counter App
# Live :https://countereactstate.netlify.app/
A simple React counter application built to practice core React concepts like **components, props, state, lifting state, and reusable UI elements**.

## Features

* Increment and decrement counter
* Reusable Button component
* State management using `useState`
* Props passing between components
* Lifting state to parent component
* Conditional logic for decrement (prevent negative values)
