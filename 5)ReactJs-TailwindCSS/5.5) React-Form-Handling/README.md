React Name List Manager
Live:-> https://reactt-formm-handlingg.netlify.app/
A simple React application demonstrating controlled forms and immutable state management by converting input values into a persistent list.
🚀 Key Features
State Immutability: Uses the spread operator ([...]) to safely update arrays.
Controlled Inputs: Synchronizes form fields with React state.
Efficient Rendering: Maps state arrays to dynamic UI elements on click.
📝 Usage
Type a name into the input field.
Click Submit to store the value in the array.
The UI automatically updates to display the new list of names.
